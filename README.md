# MSPM0G3507 PID 智能控制系统 (V2 增强版)

本仓库包含了基于 TI **MSPM0G3507** 微控制器的智能循迹小车底盘与 AI 云台追踪联动控制系统。

## 📁 目录结构 (Directory Structure)

为避免与您 GitHub 仓库原有的文件发生冲突，我们将本次升级的完整工程和代码整理到了独立的 `V2/` 目录下，实现了两代版本的完美并存：

- **`README.md`**：本说明文件（位于仓库根目录）。
- **`V1 原始配置`**（仓库根目录下的 `.project`, `.cproject` 等）：保留了原有的 V1 基础配置。
- **`V2/`**（本次核心升级目录）：包含 **V2 增强版** 的完整独立工程。
  - **`V2/main.c`**：主控调度与战术仪表盘的核心任务管理。
  - **`V2/Hardware/`**：核心底层硬件驱动及控制算法。
    - **`ctrl.c` & `ctrl.h`**：**【V2 核心大脑】** 底盘任务与云台任务的高响应多任务联动控制中心。
    - **`pid.c` & `pid.h`**：位置式与增量式 PID 计算引擎。
    - **`motor.c`**：双路电机驱动与双向静摩擦绝对补偿。
    - **`gimbal.c`**：舵机云台驱动控制。
    - **`k230.c`**：与 K230 AI 智能摄像头的高速串口通信协议解析。
    - **`track.c`**：8 路红外反射循迹传感器高速读取。
    - **`oled_hardware_i2c.c`**：OLED 硬件 I2C 驱动与战术仪表盘交互界面。
  - **`V2/empty.syscfg`**：TI 官方外设图形化配置文件。

## 🚀 V2 增强版核心升级说明 (V2 Enhanced Features)

相比于 V1 基础版，**V2 增强版** 对系统架构与核心控制算法进行了全面且深度的重构与调优：

### 1. 🧠 双线核脑联动控制架构 (`ctrl.c` / `ctrl.h`)
- **集中式调度**：将原先分散在各外设模块中的业务逻辑完全解耦，重构为集中式控制大脑 `Ctrl_Chassis_Process()`（底盘任务）与 `Ctrl_Gimbal_Process()`（云台任务）。
- **双线并发执行**：底盘高速精密循迹与云台高爆发“锁头”算法完全并行，实现了运动状态下的高精度动态目标咬合。

### 2. 🎯 云台视觉追踪算法颠覆式调优
- **极速响应（低通滤波系数暴增至 0.85）**：大幅削减软件平滑滤波带来的滞后，允许视觉模块 85% 的实时数据直接驱动电机，云台追踪延迟瞬间消失！
- **云台 X 轴分段式变参 PID 算法**：
  - **甩枪区（靶心偏离 > 40 像素）**：瞬间激活“强力锁头”，将 $K_p$ 提升至 `16.0`，配合高比例 $K_d = 4.0$ 制动，让云台以极高的角加速度甩过去；
  - **狙击区（靶心进入中心区域）**：平滑恢复为高稳态、零抖动的神仙参数（$K_p = 9.0, K_d = 2.5$），配合 `±8.0` 的高灵敏死区过滤，死死咬住目标，不产生丝毫震颤。
- **解除角速度封印**：云台输出限幅由旧版的 `150` 彻底释放至 `600`（原先的 4 倍），赋予云台瞬间爆发出最大扭矩的物理权限。

### 3. 🚗 底盘狂暴循迹与动态制动系统
- **全直道自适应测距与终点冲刺刹车算法**：高精度检测起点到首弯的脉冲用时，并在终点归位冲刺阶段自适应调整脉冲，实现**完美的压线即刻死刹**。
- **高速强力动态制动（提速 8 倍）**：在 RACING（狂暴）模式下，车辆以高基速疾驰，一旦循迹红外触发弯道信号，系统瞬间施加 `-0.40` 的反向制动力，强行在冲出赛道前将车速拽回弯道安全基速。
- **转向封印解锁**：弯道差速极限放宽至 `4.5`，允许外侧轮全力加速、内侧轮倒转逆推，强行把重车“甩”过直角弯。
- **全向静摩擦力绝对补偿**：正转与反转方向均配置独立静摩擦阶跃补偿（左轮 `24.0/48.0`，右轮 `-24.0/-48.0`），彻底消灭低占空比下电机“啸叫却不转动”的行业顽疾。

### 4. 📺 战术仪表盘 UI 与极速运行模式
- **双 UI 仪表盘自适应切换**：支持 NORM 循迹与 COMP 综合联调两种 UI 界面。
- **在线参数配置**：支持通过按键在线微调目标圈数（支持 0.5 圈微调）、运行模式切换及一键紧急启停。
- **总线安全机制**：智能车在 RUN 状态下，系统自动断开 OLED 的 I2C 刷新，将宝贵的 CPU 时钟周期和总线带宽完全释放给循迹中断与串口视觉解析，确保极速运行不卡顿。
